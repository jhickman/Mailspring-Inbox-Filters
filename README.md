## Unified Inbox Filters for Mailspring

Exclude selected accounts from the unified "All Accounts" views while keeping them visible and accessible in each account's folder tree. The plugin adds filtered unified items (Inbox, Unread, Starred, Drafts) and a Preferences tab to choose excluded accounts and set a default filtered inbox.

## Installing

1. Clone this repository or download it as a zip.
2. Install dependencies: `npm install`.
3. Build the plugin: `npm run build`.
4. In Mailspring, choose `Developer > Install a Plugin...` and select this folder.
5. Restart Mailspring or reload the main window.

## Build / Contribute

1. Clone the project.
2. Install `node_modules` with `npm install`.
3. Make changes in `src/`.
4. Compile to `lib/` with `npm run build` before testing in Mailspring.
